--------------------------------------------------------------------------------------------------------
						INFORMAÇÕES UTEIS
--------------------------------------------------------------------------------------------------------
1 - O SICOOB setup cria as pastas necessarias e instala os pacotes que serão usado
	elas serão criadas no desktop do usuario atual.

2 - O arquivo converte vai ler, estruturar e converter todos os dados dos extratos em .pdf colocados dentro da
    pasta \SICOOB\PDF\ que foi criada no desktop

3 - As duas versões do converte dizem respeito às datas dos extratos que serão convertidos
    caso os extratos sejam de anos diferentes o código pede que para cada extrato seja digitado um ano que 
    corresponde ao ano que o extrato cobre. O mês não é necessario.
